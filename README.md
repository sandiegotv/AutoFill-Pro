🚀 AutoCadastro Pro

Sistema de automação inteligente para preenchimento de dados em sistemas que não possuem integração com Excel.

---

🧠 Sobre o projeto

O AutoCadastro Pro foi desenvolvido para eliminar tarefas repetitivas de digitação manual, automatizando o processo de cadastro em sistemas através da leitura de planilhas Excel.

Ideal para empresas que utilizam sistemas antigos ou plataformas que não oferecem importação de dados, o aplicativo atua como uma ponte entre o Excel e o sistema, realizando o preenchimento automático de forma rápida e precisa.

---

⚡ Principais funcionalidades

- 📂 Importação de dados via Excel (.xlsx)
- 🤖 Automação de preenchimento com simulação de teclado
- 🖥️ Interface moderna e intuitiva
- 🔐 Sistema de licença integrado
- ▶️ Execução com um clique
- ⛔ Controle de parada da automação
- 📊 Suporte a múltiplas linhas e colunas

---

🎯 Casos de uso

- Cadastro de produtos em sistemas de e-commerce
- Registro de clientes em ERPs
- Inserção de dados em sistemas administrativos
- Automação de processos repetitivos em empresas

---

💡 Benefícios

- ⏱️ Redução de horas de trabalho manual
- ❌ Diminuição de erros de digitação
- 📈 Aumento de produtividade
- 💰 Economia de custos operacionais

---

🛠️ Tecnologias utilizadas

- Python
- CustomTkinter
- PyAutoGUI
- Pandas

---

▶️ Como executar

1. Clone o repositório
2. Instale as dependências:

pip install -r requirements.txt

3. Execute o projeto:

python app.py

---

📦 Distribuição

O sistema também pode ser disponibilizado como executável (.exe) para uso direto em ambientes Windows.

---

⚠️ Observações

- A automação depende da posição do cursor e do layout do sistema utilizado
- Recomenda-se testes antes da utilização em produção
